# Install
1. Clone the project locally:
   ```bash
   git clone https://github.com/RX-Zhang/TypeRepair.git
   ```
2. Install dependencies
   ```bash
   pip install -r requirements.txt
   ```
4. run program
   ```bash
   python main.py --dataset arg1 --sample arg2 --model arg3
   ```
